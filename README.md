# UK Job Market Dashboard (Power BI)

This project analyses over 15,000 UK job listings to explore job availability, salary trends and how opportunities vary across different sectors and regions.

As an aspiring data apprentice, I worked with a real-world dataset and followed the full data analysis process from data cleaning to building an interactive Power BI dashboard.
